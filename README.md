# Face Scan

A Python-based face detection and facial landmark recognition project using OpenCV and MediaPipe.

## Overview

**yuztarama** provides multiple implementations for detecting and analyzing human faces in images and video streams. The project includes Haar Cascade-based detection and advanced facial landmark detection using MediaPipe's FaceMesh technology.

## Features

- 🎯 **Face Detection**: Using Haar Cascade Classifier for robust face detection
- 🎭 **Facial Landmarks**: Advanced facial mesh detection with 468 3D face landmarks using MediaPipe
- 📷 **Image Processing**: Real-time face detection from static images and webcam feeds
- 🔧 **Multiple Implementations**: Different approaches for various use cases
